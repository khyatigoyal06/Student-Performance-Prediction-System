The Student Performance Prediction System is a machine-learning–based web application built to analyze student engagement data and predict their performance level using multiple ML models.

This project provides:

✔ Accurate predictions using 5 ML models
✔ A beautiful interactive React dashboard
✔ Visual comparison of predictions
✔ Insights section for interpreting results
✔ Clean Flask backend with preprocessing & inference

This project is fully deployable and perfect for academic submission or portfolio use.

🧠 Machine Learning Models Used

The system trains & compares the following algorithms:

Model	Purpose
Decision Tree	Baseline interpretability
Random Forest	High accuracy via ensembles
Perceptron	Simple linear classifier
Logistic Regression	Probabilistic prediction
MLP Classifier	Deep learning (neural network)

Each model outputs a performance label:
H : High Performer
M : Medium Performer
L : Low Performer
